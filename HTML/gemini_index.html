1. The Basics

HTML (HyperText Markup Language) is the backbone of the web. It defines the structure of a webpage.

The Boilerplate

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>

5. Semantic HTML5

Semantic tags describe their meaning to both the browser and the developer (Better SEO and Accessibility).

  • <header>: Container for introductory content.
  • <nav>: Navigation links.
  • <main>: Main content of the document.
  • <article>: Independent, self-contained content.
  • <section>: Thematic grouping of content.
  • <aside>: Content aside from the main content (sidebars).
  • <footer>: Footer for its nearest sectioning content.

7. Advanced HTML Concepts

Iframe

Embedding another document within the current HTML document.

<iframe src="https://example.com" title="Iframe Example"></iframe>

The <head> and Meta Tags

Information about the page that isn't displayed.

<meta name="description" content="Free HTML tutorial">
<meta name="keywords" content="HTML, CSS, JavaScript">
<meta name="author" content="John Doe">

Data Attributes

Used to store custom data private to the page or application.

<div id="user" data-id="123" data-role="admin">...</div>

Canvas & SVG

Canvas is for scriptable (JS) 2D/3D graphics, SVG is for XML-based vector graphics.
