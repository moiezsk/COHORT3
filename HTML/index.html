HTML BASICS

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages and web applications. HTML describes the structure of a web page using markup. It consists of a series of elements, which you use to enclose or wrap different parts of the content to make it appear a certain way, or act a certain way.

Text and Content Tags

HTML provides various tags to structure and format text and content on a web page. Some of the commonly used text and content tags include:

  • <h1> to <h6>: Heading tags used to define headings of different levels.
  • <p>: Paragraph tag used to define a block of text.
  • <ul>: Unordered list tag used to create a bulleted list.
  • <ol>: Ordered list tag used to create a numbered list.
  • <li>: List item tag used within <ul> or <ol> to define individual list items.
  • <a>: Anchor tag used to create hyperlinks.
  • <img>: Image tag used to embed images in a web page.

Formatting Tags in HTML

HTML provides various formatting tags to style and format text on a web page. Some of the commonly used formatting tags include:

  • <strong>: Used to indicate strong importance or emphasis on text.
  • <em>: Used to indicate emphasis on text, often displayed in italics.
  • <u>: Used to underline text.
  • <br>: Line break tag used to create a new line without starting a new paragraph.
  • <hr>: Horizontal rule tag used to create a thematic break or divider between sections of content.

Media Tags in HTML

HTML provides various media tags to embed and display multimedia content on a web page. Some of the commonly used media tags include:

  • <audio>: Used to embed audio content, allowing users to play and control audio files.
  • <video>: Used to embed video content, allowing users to play and control video files.
  • <iframe>: Used to embed another HTML document within the current document, often used for embedding videos or maps.

Table Tags in HTML

HTML provides various table tags to create and structure tabular data on a web page. Some of the commonly used table tags include:

  • <table>: Used to define a table and its structure.
  • <tr>: Table row tag used to define a row in a table.
  • <th>: Table header tag used to define a header cell in a table, typically displayed in bold and centered.
  • <td>: Table data tag used to define a standard cell in a table, containing data or content.

Form Tags in HTML

HTML provides various form tags to create interactive forms for user input on a web page. Some of the commonly used form tags include:

  • <form>: Used to define a form and its attributes, such as action and method.
  • <input>: Used to create various types of input fields, such as text, password, checkbox, radio button, etc.
  • <textarea>: Used to create a multi-line text input field for longer user input.
  • <select>: Used to create a drop-down list of options for users to choose from.
  • <button>: Used to create clickable buttons for form submission or other actions.








Fieldset and Legend Tags in HTML

The <fieldset> tag is used to group related elements within a form, while the <legend> tag is used to provide a caption or title for the fieldset. These tags help improve the organization and accessibility of forms by visually grouping related form controls together.

Semantic Tags in HTML

Semantic tags in HTML are used to provide meaning and structure to the content of a web page. These tags help search engines and assistive technologies understand the content better. Some commonly used semantic tags include:

  • <header>: Represents the introductory content or a group of navigational links.
  • <nav>: Represents a section of navigation links.
  • <main>: Represents the main content of the document.
  • <article>: Represents a self-contained piece of content, such as a blog post or news article.
  • <section>: Represents a thematic grouping of content, typically with a heading.
  • <aside>: Represents content that is tangentially related to the main content, such as sidebars or pull quotes.
  • <footer>: Represents the footer of a document or section, typically containing information about the author, copyright, or related links.
