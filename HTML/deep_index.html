🌱 Level 1: Essential HTML & Document structure

fundamental tags

📄 Doctype & Core skeleton

Every HTML document begins with <!DOCTYPE html> followed by <html>, <head>, <body>. Meta tags control viewport & character set.
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document structure</title>
</head>
<body>
  <h1>Hello HTML</h1>
  <p>This is a basic page skeleton.</p>
</body>
</html>

🔤 Headings & Paragraphs

<h1>...<h6> for hierarchical titles, <p> for text blocks. <br> line break, <hr> horizontal rule.
<h1>Main title</h1>
<h2>Subheading</h2>
<p>This is a paragraph with <strong>bold</strong> and <em>italic</em>.<br>Line break here.</p>
<hr>

🔗 Links & Images

<a> anchor tags create hyperlinks, <img> embeds images (self-closing). Attributes: href, src, alt, target.
<a href="https://example.com" target="_blank">Visit Example</a>
<img src="https://picsum.photos/id/1/150/100" alt="Random mountain" width="150" height="100">

🗂️ data-* attributes & output

Custom data attributes store extra info. <output> for calculation results.
<div data-user-id="12345" id="user">John Doe</div>
<form oninput="result.value = parseInt(a.value) + parseInt(b.value)">
<input type="range" name="a" value="50"> + <input type="number" name="b" value="10">
= <output name="result">60</output>
</form>
